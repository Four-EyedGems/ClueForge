# 🧩 ClueForge

**ClueForge** is an offline escape-room generator that turns any room into an interactive puzzle experience.
Scan real-world objects, generate chained riddles, and create a complete escape-room game in minutes.

---

## 🚀 Overview

ClueForge allows you to:

* 📸 Scan objects using your device camera
* 🧠 Generate riddles based on detected objects
* 🔗 Create a chained puzzle experience
* 🖨️ Print or display QR-based clue cards
* 🎮 Play with friends in a real-world escape-room setup

All of this works **fully offline**, ensuring speed, privacy, and reliability.

---

## 🔒 Privacy-First Design

ClueForge is built with a strong focus on user privacy:

* ❌ No account required
* ❌ No personal data collection (name, email, phone, location)
* 📱 All data stays **on your device**
* 📷 Camera is used only for **on-device object detection**
* 🌐 No game data is sent to any server

The only external integration is **Google AdMob** for ads.

👉 See full details in [Privacy Policy](./privacy-policy.html)

---

## 📱 Key Features

### 🎯 Offline Gameplay

* Works without internet after installation
* No cloud dependency

### 🤖 On-Device AI

* Uses TensorFlow Lite (MobileNet)
* Object detection happens locally

### 🧩 Dynamic Puzzle Generation

* Creates unique riddle chains every session
* Built-in puzzle database

### 🎨 Custom Experience

* Themes, difficulty levels, and settings
* Adjustable gameplay preferences

### 📦 Local Storage

* Game state and settings stored securely on-device
* No external storage or sync

## 💰 Monetization

ClueForge is **free to use** and supported by ads via Google AdMob:

* Banner ads (non-game screens)
* Interstitial ads (limited frequency)
* Optional rewarded ads (for hints)

Users can:

* Choose personalized or non-personalized ads
* Manage consent anytime inside the app
